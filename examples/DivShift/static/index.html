DivShift NAWC bias dashboard

Bias is the interface.

This dashboard scans divshift_nawc.csv, maps each row back to its photo inside the nested state folders, and turns the existing partitions into a browsable atlas of spatial, temporal, taxonomic, sociopolitical, and observer-driven bias.
