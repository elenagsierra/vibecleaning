vibecleaning

Core Scaffold

Generic lineage and execution APIs for self-contained local apps.

This server exposes the reusable Vibecleaning core: project state, dataset lineage, artifact access, previews, analyses, steps, head changes, and undo. Example applications now ship their own frontends and routes as self-contained apps.

Run the generic scaffold python server.py
Run the trajectory example app python examples/trajectory/server.py
